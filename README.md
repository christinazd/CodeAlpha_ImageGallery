CodeAlpha Image Gallery

A modern, responsive Image Gallery built using HTML, CSS, and JavaScript that allows users to browse images interactively with smooth navigation and enhanced UI effects.

This project was developed as part of the CodeAlpha Frontend Development Task.

 Features

 Grid-based Image Gallery Layout

 Lightbox View for enlarged image display

 Next / Previous Navigation Controls

 Hover Effects & Smooth Transitions

 Fully Responsive Design (Mobile / Tablet / Desktop)

 Clean and Interactive User Interface

 Technologies Used

HTML5

CSS3 

JavaScript 

 Functionality

The gallery allows users to:

View images in a structured layout

Click any image to open it in Lightbox Mode

Navigate between images using:

Next button

Previous button

Experience smooth UI animations

Browse comfortably across all screen sizes
